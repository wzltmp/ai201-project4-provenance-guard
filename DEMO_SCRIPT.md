> **Note:** `confidence` will be somewhere in the **0.60–0.75** range. `attribution` should

> visibly lean AI, which is fine to narrate.

**SAY:**
> "Classic AI marketing copy — 'leverage,' 'holistic,' 'transformative.' All three signals
> agree: the LLM flagged generic structure, stylometry caught uniform sentence lengths and
> AI vocabulary, and the readability signal found zero personal pronouns. That agreement is
> what drives a confident verdict. Look at the label text — it calls itself an estimate
> and offers an appeal, even at high confidence."

## SCENE 4 — Borderline human submission

*Output appears automatically as Step 3 of the script.*

**YOU'LL SEE:**
```json
{
    "content_id": "c_yyyyyyyyyyyy",
    "attribution": "uncertain",
    "p_ai": 0.65,
    "confidence": 0.22,
    "signals": {
        "llm":         { "p_ai": 0.80, "rationale": "Well-structured, lacks personal voice..." },
        "stylometric": { "p_ai": 0.35, "features": { "burstiness": 0.26, "ai_vocab_hits": 0 } },
        "readability": { "p_ai": 0.75, "features": { "pronoun_density": 0.0, ... } }
    },
    "label": {
        "variant": "uncertain",
        "text": "🤔 Origin unclear. Provenance Guard could not confidently determine
                 whether this was written by a person or generated by AI (22% confidence).
                 Treat this as inconclusive. If you're the creator, you can add context
                 by appealing."
    },
    "status": "classified"
}
```

> **Note:** `attribution` will always be `"uncertain"` here. `confidence` will be in the
> **0.15–0.30** range. The key thing to point at: LLM is ~0.80, stylometric is ~0.35 —
> that gap is what collapses confidence.

**SAY:**
> "Economics paragraph — polished, formal, but no AI vocabulary and natural sentence
> variation. The LLM flagged the formal structure as AI-like, around 0.80. Stylometry
> looked at sentence variance and found no AI fingerprints, around 0.35. That 0.45 gap
> between signals triggers the disagreement penalty in the fusion formula. Confidence
> collapses to around 22% — well below our 0.70 bar. The system correctly refuses to
> accuse. The label says 'origin unclear' and offers the appeal path."

## Quick reference — what each verdict means

| What you see | Why |
|---|---|
| `"attribution": "ai"`, confidence 0.60–0.75 | All 3 signals agree, clears the 0.70 bar |
| `"attribution": "uncertain"`, confidence 0.15–0.30 | LLM/stylometry gap triggers disagree penalty |
| `"attribution": "uncertain"`, low p_ai | Signals lean human but text is too short to trust |

## If something goes wrong

| Problem | Fix |
|---|---|
| Step 2 returns `"uncertain"` instead of `"ai"` | LLM had a softer run — narrate "signals still visibly lean AI" and continue |
| 429 error mid-demo | You've run it too many times in the last minute — wait 60s |
| `content_id not found` on the appeal step | Demo script captures the ID automatically; only happens if step 3 failed |
